Боты

Вся стуктура бота

pBot - Возращается любым ивентом. Это сам бот.

pBot.getNickname() : (String) - Никнейм бота.

pBot.jump() : (Void) - Заставляет подпрыгнуть бота.

pBot.stopBot() : (Void) - Отключить и остановить бота.

pBot.hasMap() : (Boolean) - Вернет true если у бота имеется карта в инвенторе.

pBot.currentWindowID() : (Integer) - Вернет ID текущего меню.

pBot.isOnline(slot:Integer) : (Void) - Установит слот в хотбаре.

pBot.dropItem(slot:Integer) : (Void) - Выкинет предмет из инвентаря.

pBot.sendMessage(msg:String) : (Void) - Отправит сообщение от бота.

pBot.sendPacket(packet:MinecraftCPacket) : (Void) - Отправит пакет на сервер.

pBot.useItem() : (Void) - Нажмет пкм по текущему предмету в хотбаре.

pBot.windowClick(slot:Integer, mouseButton:Integer) : (Void) - Нажмет на предмет в текущем инвентаре.

pBot.windowTitle : (String) - Вернет название текущего меню.

------------------------------------------------------------

pBot.getWorld() : (PBotWorld) - Мир, в котором находится бот.

pBot.getWorld().loadedEntityList : (List <>Entity>) - Список всех сущностей в мире.

pBot.getWorld().getBlockState(BlockPos) : (IBlockState) - Получить блок по координатам.

pBot.getWorld().getBlockState(BlockPos) : (IBlockState) - Получить блок по координатам.

------------------------------------------------------------

pBot.player : (PBotEntity) - Информация бота, координаты, инвентарь и т.д.

pBot.player.openContainer : (Container) - Инвентарь бота.

pBot.player.openContainer.getInventory() : (List <>ItemStack>) - Список предметов в инвентаре.

pBot.player.openContainer.windowId : (Integer) - Вернет ID текущего меню.

pBot.player.player.posX : (Double) - Координата X бота.

pBot.player.player.posY : (Double) - Координата Y бота.

pBot.player.player.posZ : (Double) - Координата Z бота.

pBot.player.player.rotationYaw : (Float) - Yaw позиция камеры бота.

pBot.player.player.rotationPitch : (Float) - Pitch позиция камеры бота.

pBot.player.player.onGround : (Boolean) - Касание бота с землей.
